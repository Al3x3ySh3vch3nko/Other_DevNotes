Для работы с sass

1) установить node.js
2) в терминале проверить установку | npm --version |
3) сохранить в папку файл package.json
3) установить для текущей папки командой | nmp init -y |
4) ctrl+l очищает строку и можно делать вставку появившегося в терминале кода
5) далее комадной | npm install node-sass |установить sass
6) в папку установятся файлы package-lock.json и node-modules
7) в коде 
"scripts": 
{
    "test": "echo \"Error: no test specified\" && exit 1"
},
прописываем типа этого
"scripts": {
    "sass": "node-sass -w scss/ -o dist/css/ --recursive"
},

там же в коде использовать название папки, которая используется в названии папки.
8) далее делать папку sass и в ней файл main.scss. Также далее - папку dist, где будет компиляция
9) далее - запустить sass в терминале | npm run sass | и запускать ее каждый раз перед началом работы или созданием нового файла
10) проверить, что код компилируется. сделать переменную и задать ее какому-то элементу в main.scss. сохранить, увидеть, что в консоли отмечена успешная компиляция и в файле css
появился правильно скомпилированный файл main.css
11) далее создать необходимые папки проекта в папке dist.
12) для импорта модулей и переменных можно использовать отдельный файл в папке scss
| _variables_scss | где первая черта означает, что файл не будет компилироваться.
При этом в sass следует добавить импорт через код | @import 'variables' ; |
13) для функций создается файл _functions.scss 
---
14) чтобы запустить функцию color в модулях следует писать атрибт как
| color: set-text-color($название переменной); |
