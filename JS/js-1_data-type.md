#### __1. Number \ Число__ : представляет числовое значение
********

__Методы чисел:__

__Number()__ : Перевести строку в число:
- через функцию Number
`Number('23')`
- через оператор сложения, при котором движоr актоматически конвертирует строку в число
`+'23'`

__Number.parsing()__ : Чтобы найти и выделить число из строки, содержащей и число и буквы
`Number.parseInt('30px', 10)` 
> вернет в формате "число" цифру 30. При этом строка должна начинаться с цифры, а не буквы, число 10 на конце означает формат вывода числа (в данном случае десятичный).
Не поддерживается парсинг числа с плавающей запятой. Для нее следует использовать
`Number.parseFloat('2.5rem')`

__Number.isFinite()__ : Проверка является ли данное числом или нет
`Number.isFinite(10)` \\\ да 
`Number.isFinite('10')` \\\ нет

__Math()__ : Основная функция для работы с числами

__Math.sqrt()__ : Квадратный корень из числа
`Math.sqrt(25)` \\\ даст 5

__Math.max()__ : (Узнать максимальное число) \ _Math.min()_ : (Узнать минимальное число)
`Math.max(5,18,45,56)`
`Math.min(5,18,45,56)`
В процессе преобразует строки в числа, но не парсинг

__MathPI()__ : Число Пи
`MathPI()`

__Math.floor(), Math.random()__ : Рандомное число и округление
`Math.floor(Math.random() * необходимое число) + 1`
> Получить число между максимальным и минимальным
`const randomInt = (min, max) => Math.trunc(Math.random() * (max - min) + 1) + min`

> Округление числа с плавающей запятой - до простого
- исключая часть после запятой в положительных и отрицательных числах
Math.trunc()

- округляя в большую или меньшую сторону сторону - до простого
`Math.round()`

- округляя всегда в сторону увеличения числа если после запятой есть какое-то значение - до простого
`Math.ceil()`

- округление числа с плавающей запятой с сохранением запятой
`(2.7)toFixed(0)` \\\ возвращает строку, второе число - количество возвращаемых знаков после запятой

- остаток от деления
`5 % 2` \\\ 1
Используется для определения четных \ нечетных чисел
Функция
const isEven = n => n % 2 === 0

- округляя всегда в сторону уменьшения числа если после запятой есть какое-то значение, убирает часть после запятой в положительных числах и округляет число в отрицательных
`Math.floor()`

__Методы строк:__
__indexOf(),lastIndexOf()__ : Получить значение по индексу
`airlines.indexOf('A')` - даст цифру 2 как индекс буквы А (первой которая попадется) 
`airlines.indexOf('TAP')` - даст цифру 1 как индекс первой буквы слова TAP 
`airlines.lastIndexOf('A')` - даст цифру 5 как индекс буквы А (последней которая попадется) Если значения не найдено, то выдает значение -1

__slice()__ : Вырезает строку по индексу 
`airlines.slice(4)`  \\\ 4 это стартовое значение, до которого будет вырезано, а с которого - оставлено.
[ ! ] Это не меняет саму строку, а возрващает новую
`airlines.slice(4, 7)` \\\ значение с 4 ДО 7 будет вырезано в новую строку.

> Чтобы вырезать первое слово, не зная индекса его букв
airlines.slice(0, airlines.indexOf (' ') ) \\\ вырезает с индекса 0 и до первого пробела, который рассчитывается как airlines.indexOf (' ').

> Чтобы вырезать последнее слово, не зная индекса его букв
`airlines.slice(0, airlines.lastIndexOf` (' ') + 1 ) \\\ вырезает с последнего индекса, который соответствует последнему пробелу и до конца и добавляет + 1 индекс чтобы сам проблем не включался.

> Если указывать индексы в этом методе с отрицательным значением он будет работать с конца строки.
К примеру, 
`airlines.slice(-2)` \\\ вырежет 2 последние буквы строки.
`дairlines.slice(1, -1)` \\\ вырежет всю строку кроме первой и последней буквы .

__toLowerCase() \ toUpperCase()__ : Увеличение \ уменьшение регистра букв
`airlines.toLowerCase()`
`airlines.toUpperCase()`
Пример - 
\\\ исправление строки с разным регистром на регистр с первой заглавной и остальными обычными
```
const passenger = 'JoNaS'
const passengerLower = passenger.toLowerCase()
const passengerCorrect = passengerLower[0].toUpperCase() + passengerLower.slice(1) 
```

__trim()__ : Убрать пробелы и знаки типа \n
`airlines.trim()`

__replace() \ replaceAll()__ : Замена значений
`const priceGB = '288,34'`
`priceGB.replace(',', '.')` \\\ первый аргумент то что меняется, второй - то на что меняется. Запятая меняется на точку.
Поддерживает цепочку методов
`priceGB.replace(',', '.').replace('34', '44')`
Меняет только первое значение!
Можно использовать 
`priceGB.replaceAll` (если он уже введен и поддерживается)
Или регулирные выражения
`priceGB.replace(/wrong word/g, 'correct word')`

__includes() \ startsWith() \ endsWith()__ : Методы, возвращающие булевые значения
const plane = 'A320nvvv'
`plane.includes('A320')` \\\ строка plane содержит слова A320 = true
`plane.startsWith('A')` \\\ строка plane начинается с буквы A = true
`plane.endsWith('320')` \\\ строка plane заканчивается цифрами 320 = true

__split()__ : Метод разделяет строку на строки и вносит каждое как элемент единого массива при вызове метода указывается символ, который разделяет строку
`'a+very+nice+string'.split('+')` \\\ создаст массив с элементами [a, very, nice, string]
Можно использовать с деструктуризацией
`const [firstName, lastName] = 'Alex Shev'.split(' ')` 

__join()__ : Противоположный метод
join('символ-разделитель')

Пример:
`const newName = ['Mr.', firstName, lastName.toUpperCase()].join(' ')`

__padding()__ : Добавляет количество символов, необходимых для того, чтобы строка имела заданное количество символов
Пример:
`const message = ' Go to gate 25'`
`message.padStart(25, '+')` \\\ добавит то количество символов + в начале строки, чтобы она имели длину 25 символов
`message.padStart(25, '+').padEnd(30, '+')` \\\ добавит еще символы + в конце, чтобы длина была 30
Пример:
заменить все цифры кредитной карты кроме последних 4 на *
```    
const  maskCreditCard = function (number)
{ 
    const str = number + ' ' \\ то же самое что и метод String
    const last = str.slice(-4) \\ вырезает 4 последние цифры
    return last.padStart(str.lenght, '*')
}
```

__repeat()__ : Повторяет строку
```
let planesInLine = function (n)
{
    console.log(`There are ${n} planes in line ${'+'.repeat(n)}`)
}
planesInLine(7)
```
